## Typescript

### `genVariable(name, value, options)`

Create variable declaration.

**Example:**

```js
genVariable("a", "2");
// ~> `const a = 2`

genVariable("foo", "'bar'");
// ~> `const foo = 'bar'`

genVariable("x", "1", { kind: "let" });
// ~> `let x = 1`

genVariable("y", "2", { export: true });
// ~> `export const y = 2`
```
