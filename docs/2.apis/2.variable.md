This module provides APIs for generating safe variable names and variable declarations (`const` / `let`) with export and kind options.

## Typescript

### `genVariable(name, value, options)`

Create variable declaration.

**Example:**

```js
genVariable("a", "2");
// ~> `const a = 2`

genVariable("foo", "'bar'");
// ~> `const foo = 'bar'`

genVariable("x", "1", { kind: "let" });
// ~> `let x = 1`

genVariable("y", "2", { export: true });
// ~> `export const y = 2`
```

### `genVariableName(name)`

Generate a safe javascript variable name.

**Example:**

```js
genVariableName("valid_import");
// ~> `valid_import`

genVariableName("for");
// ~> `_for`

genVariableName("with space");
// ~> `with_32space`
```
