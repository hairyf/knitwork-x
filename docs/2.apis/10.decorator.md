## Typescript

### `genDecorator(name, args?, indent)`

Generate decorator: `@decorator` or `@decorator(args)`.

**Example:**

```js
genDecorator("Component");
// ~> `@Component`

genDecorator("Injectable", "()");
// ~> `@Injectable()`

genDecorator("Route", '("/api")');
// ~> `@Route("/api")`

genDecorator("Validate", "(min: 0, max: 100)");
// ~> `@Validate(min: 0, max: 100)`
```
