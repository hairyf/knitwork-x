## Typescript

### `genConditionalType(checkType, extendsType, trueType, falseType)`

Generate conditional type.

**Example:**

```js
genConditionalType("T", "U", "X", "Y");
// ~> `T extends U ? X : Y`

genConditionalType("T", "null", "never", "T");
// ~> `T extends null ? never : T`
```
